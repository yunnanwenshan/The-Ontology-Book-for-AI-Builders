# 附录 C · 学习计划（按周）

> 16-21 小时投入，2-3 周完成。

## 总体目标

- 能独立设计、建模、查询、部署一个本体驱动项目
- 能向团队讲清"为什么选这个不选 RAG"
- 能跑通一个完整 demo

## 自评 Checklist

学完后能打勾：

- [ ] 能用 1 句话说出"什么是本体"
- [ ] 能用 5 句话说"为什么 AI 需要本体"
- [ ] 能写出一个完整 RDF 三元组
- [ ] 能区分 OWL 5 个类构造器、5 个属性特征
- [ ] 能写出 5 种 SPARQL 查询
- [ ] 能写出 5 种 SHACL 约束
- [ ] 能用 Protégé 打开一个本体
- [ ] 能启动 Fuseki，跑第一个 SPARQL
- [ ] 能用 R2RML 把 SQL 转 RDF
- [ ] 能用 MCP 把本体接 Agent
- [ ] 能用 1 张表讲清本体 vs RAG vs GraphRAG

**全打勾 = 你可以写《AI 本体论实战》这本书了。**

## 三条学习路径

### 路径 A · 周末突击（4 小时）

**适合**：想知道"对我有没有用"

- 第 1 天（2h）：`03-engineering/01-hello-world.md` 跑通 demo
- 第 2 天（2h）：`02-specs/03-sparql.md` 学 5 个查询模式

**产出**：能跑 Fuseki + 写出 5 个 SPARQL

### 路径 B · 系统学习（3 周，16-21 小时）

**产出**：能独立搭系统

### 路径 C · 老板速览（30 分钟）
