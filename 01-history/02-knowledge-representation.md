# 1.2 · 知识表示 60 年演变（速读版）

> **一句话**：从"画图"到"标准语言"，人类花 60 年让机器"懂"知识。

## 1 句话总结

```
画图（1968）→ 框架（1975）→ 推理机（1985）→ 语义网（1998）→ RDF/OWL（2004）→ SHACL（2017）→ LLM+本体（2024）
```

## 5 个里程碑

| 年 | 事件 | 解决的问题 |
|----|------|-----------|
| **1968** | Quillian 语义网络 | "概念可以画成图" |
| **1985** | KL-ONE（带推理的本体）| "机器能自己推出新事实" |
| **1998** | Tim Berners-Lee 提出语义网 | "互联网上数据要机器可读" |
| **2004** | RDF/OWL 成 W3C 标准 | "跨工具、跨公司、互操作" |
| **2017** | SHACL | "业务规则也能机器校验" |

## 为什么 2025 年又火

```
LLM 幻觉严重  →  需要"事实核查层"  →  本体
LLM + 图 = GraphRAG
MCP 协议       →  Agent 能直接调本体
```

**本体的角色变了**：从"独立知识库" → "AI 的脚手架"。
