# 实战案例 2 · 客服系统本体

> **场景**：电商客服系统，订单/物流/退货/工单一图全关联
> **难度 / 时间**：⭐ 入门 / 30 分钟
> **谁在用**：客服主管看仪表盘 / 客服问 Claude / 开发者跑 cs-query.py

> **场景**：电商客服系统，订单/物流/退货/工单一图全关联
> **目标**：AI 客服 1 秒回答常见问题
## 1. 业务问题

客服每天问：
- "客户 001 最近订单是什么？"
- "iPad 的物流到哪了？"
- "客户 001 的退货进度？"
- "哪些工单还没处理？按优先级排"
- "工单分布"
- "VIP 客户是谁？"

## 2. 跑 demo

```bash
cd ~/Documents/projects/ai/ontology/05-real-cases/customer-service
```

## 3. 业务价值

| 不用本体 | 用本体 |
|---------|--------|
| 客服翻 5 个 Excel | 一句话查出来 |
