## 第 2 步：看懂第一个查询（1 分钟）

```sparql
PREFIX ex: <http://example.com/shop#>
SELECT ?o ?amount WHERE {
    ?o a ex:Order ;
       ex:amount ?amount .
} ORDER BY DESC(?amount)
```

读法：
- `PREFIX` 声明前缀
- `SELECT ?o ?amount` 要这两列
- `?o a ex:Order` 找所有类型是 Order 的（`a` = `rdf:type`）
- `;` 还是这个主语
- `ORDER BY DESC(?amount)` 按金额倒序

## 第 4 步：5 种查询模式（5 分钟）

打开 `examples/7-shop.py` 看 3 个查询。

### Q1: 基本（找所有订单）
```sparql
SELECT ?o ?amount WHERE {
    ?o a ex:Order ;
       ex:amount ?amount .
} ORDER BY DESC(?amount)
```

### Q2: 属性匹配（Alice 买过什么）
```sparql
SELECT ?product ?price WHERE {
    ?order ex:customer ex:Alice ;
           ex:item ?product .
    ?product ex:price ?price .
}
```

### Q3: 聚合 + HAVING（VIP 客户）
```sparql
SELECT ?name (SUM(?amount) AS ?total) WHERE {
    ?order ex:customer ?cust ;
           ex:amount ?amount .
    ?cust ex:name ?name .
} GROUP BY ?cust ?name
HAVING (SUM(?amount) > 5000)
```

### Q4: FILTER（金额 > 7000 的订单）

加到 `examples/7-shop.py` 末尾：

```python
q4 = prepareQuery("""
PREFIX ex: <http://example.com/shop#>
SELECT ?o ?amount WHERE {
    ?o a ex:Order ;
       ex:amount ?amount .
    FILTER (?amount > 7000)
}
""")
print("\nQ4: 金额超 7000 的订单")
for row in g.query(q4):
    print(f"  {row.o.split('#')[-1]}  ¥{row.amount}")
```

跑：`python3 examples/7-shop.py`，应该看到 o2 一条。

### Q5: OPTIONAL（订单和客户，有 phone 就显示）

```python
# 临时给 Alice 加个 phone
import rdflib
g.add((rdflib.URIRef("http://example.com/shop#Alice"),
       rdflib.URIRef("http://example.com/shop#phone"),
       rdflib.Literal("13800000000")))

q5 = prepareQuery("""
PREFIX ex: <http://example.com/shop#>
SELECT ?name ?phone WHERE {
    ?cust a ex:Customer ;
          ex:name ?name .
    OPTIONAL { ?cust ex:phone ?phone . }
}
""")
print("\nQ5: 客户 + phone（没有的也显示）")
for row in g.query(q5):
    phone = row.phone if row.phone else "（无）"
    print(f"  {row.name}  {phone}")
```
