## 第 2 步：看代码（30 秒）

打开 `examples/1-first-rdf.py`：

```python
from rdflib import Graph, URIRef, Literal, Namespace
from rdflib.namespace import RDF, RDFS

g = Graph()                              # 1. 建一个空图
EX = Namespace("http://example.com/")    # 2. 定义命名空间

g.add((EX.Alice, RDF.type, EX.Person))   # 3. 加 4 条三元组
g.add((EX.Alice, EX.age, Literal(30)))
g.add((EX.Alice, EX.knows, EX.Bob))
g.add((EX.Bob, RDF.type, EX.Person))
```

**3 个关键点**：
- `g = Graph()` — 创建空图
- `Namespace("http://example.com/")` — 定义前缀
- `g.add((s, p, o))` — 加三元组

## 第 4 步：看代码（30 秒）

```python
import csv
from rdflib import Graph, Literal, Namespace
from rdflib.namespace import RDF

g = Graph()
EX = Namespace("http://example.com/people/")

with open("data/people.csv") as f:    # 1. 读 CSV
    reader = csv.DictReader(f)
    for row in reader:                # 2. 逐行处理
        person = EX[row["name"].replace(" ", "_")]
        g.add((person, RDF.type, EX.Person))   # 3. 加三元组
        g.add((person, EX.name, Literal(row["name"])))
        g.add((person, EX.email, Literal(row["email"])))
        g.add((person, EX.company, Literal(row["company"])))
```
