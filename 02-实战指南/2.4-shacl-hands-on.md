## 第 3 步：看代码（1 分钟）

`examples/3-shacl.py`：

```python
from pyshacl import validate
from rdflib import Graph

shapes = Graph().parse("data/shapes.ttl", format="turtle")
data   = Graph().parse("data/bad-data.ttl", format="turtle")

conforms, report_graph, _ = validate(data, shacl_graph=shapes, inference="rdfs")
print(f"通过？ {conforms}")

# 解析报告
import re
report_str = report_graph.serialize(format="turtle")
matches = re.findall(
    r'sh:focusNode\s+(\S+).*?sh:resultMessage\s+"([^"]+)"',
    report_str, re.DOTALL
)
for node, msg in matches:
    print(f"  {node}: {msg}")
```

## 第 4 步：自己写一个 SHACL（5 分钟）

打开 `data/shapes.ttl` 加一条规则：

```turtle
@prefix sh: <http://www.w3.org/ns/shacl#> .
@prefix ex: <http://example.com/shop#> .
@prefix xsd: <http://www.w3.org/2001/XMLSchema#> .

ex:OrderShape
    a sh:NodeShape ;
    sh:targetClass ex:Order ;
    sh:property [
        sh:path ex:customer ;
        sh:minCount 1 ;
        sh:class ex:Customer ;
    ] , [
        sh:path ex:amount ;           # ← 新增
        sh:datatype xsd:decimal ;     # ← 新增
        sh:minInclusive 0 ;           # ← 新增
    ] .
```

跑 `3-shacl.py` 还能抓到 `order-001` 缺 customer。如果想测金额校验，把 `bad-data.ttl` 改成：

```turtle
@prefix ex: <http://example.com/shop#> .
@prefix xsd: <http://www.w3.org/2001/XMLSchema#> .

ex:Alice a ex:Customer .

ex:order-002 a ex:Order ;
    ex:customer ex:Alice ;
    ex:amount -50.00 .   # 负数
```

跑 `3-shacl.py`，应该看到"金额负数"的违规。

## 第 5 步：练一下（5 分钟）

**文件位置**：
- `04-zero-install-demo/data/my-shapes.ttl`（SHACL 规则）
- `04-zero-install-demo/data/my-data.ttl`（测试数据）

新建 `data/my-shapes.ttl`，要求：

1. 每个 Product 必须有 name
2. price 必须 >= 0
3. phone 字段必须是"1"开头的 11 位数字（用 `sh:pattern`）

```turtle
@prefix sh: <http://www.w3.org/ns/shacl#> .
@prefix ex: <http://example.com/shop#> .
@prefix xsd: <http://www.w3.org/2001/XMLSchema#> .

ex:ProductShape
    a sh:NodeShape ;
    sh:targetClass ex:Product ;
    sh:property [
        sh:path ex:name ;
        sh:minCount 1 ;
    ] , [
        sh:path ex:price ;
        sh:minInclusive 0 ;
    ] .
```

新建 `data/my-data.ttl`：

```turtle
@prefix ex: <http://example.com/shop#> .
@prefix xsd: <http://www.w3.org/2001/XMLSchema#> .

# 故意有 3 处违规
ex:good a ex:Product ; ex:name "好商品" ; ex:price 100 .
ex:no-name a ex:Product ; ex:price 200 .         # 缺 name
ex:negative a ex:Product ; ex:name "负价" ; ex:price -10 .  # 负价格
```

跑：

```python
from pyshacl import validate
from rdflib import Graph

shapes = Graph().parse("data/my-shapes.ttl", format="turtle")   # 路径是相对当前执行目录
data   = Graph().parse("data/my-data.ttl",   format="turtle")

conforms, report, _ = validate(data, shacl_graph=shapes)
print(f"通过？{conforms}")
print(report.serialize(format="turtle"))
```

应该抓到 2 条违规。
